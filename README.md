# Raytracing in One Weekend in WebGPU

This is a port of the code in the book [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) to WebGPU.

## Running
```sh
bun install
bun run dev
```
